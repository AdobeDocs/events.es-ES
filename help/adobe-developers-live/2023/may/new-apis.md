# Visita a la API: Conozca las nuevas API que enviamos en los últimos 6 meses

Visita a la API: Conozca las nuevas API que enviamos en los últimos 6 meses

Esta charla trata sobre la transición en curso a una arquitectura sin encabezado para AEM. El nuevo enfoque se centrará en permitir que los usuarios realicen todas las acciones a través de llamadas de API simples pero potentes, teniendo en cuenta la velocidad del proceso. Las dos API que le guiaremos son las API del plano de control Modelos de fragmentos de contenido (CFM) y Fragmentos de contenido (CF). Con estas dos API, los usuarios pueden crear fácilmente modelos complejos y rellenarlos de una manera rápida y directa. Se le guiará a través de la creación de un nuevo modelo de fragmento de contenido y, a continuación, rellenando fragmentos de contenido basados en él. A continuación, actualizaremos el modelo y crearemos otros fragmentos basados en la nueva estructura.
