# Uso de Workfront para informes

Descubra cómo Workfront mejora la reestructuración organizativa administrando los flujos de trabajo de forma eficaz. Este taller proporciona información sobre el uso de Workfront para gestionar los cambios en equipos y funciones, lo que garantiza transiciones sin problemas y mantiene la productividad. Aprenda estrategias para optimizar la instancia de Workfront durante los informes, incluida la auditoría de funciones de usuario, el ajuste de permisos y el uso de herramientas como modelos y promoción de entornos. Este conocimiento le permite mantener sus proyectos en curso, a pesar de los cambios organizativos.
