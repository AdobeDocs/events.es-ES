# Nathan&#39;s Five Faves: API básicas para el mundo de Workfront

¿Quiere saber qué puede hacer realmente la API de Workfront por usted? Únase a Nathan Johnson, ingeniero de soporte técnico de Adobe III, mientras desglosa los aspectos básicos y lleva las API al mundo real.

En esta sesión, Nathan analiza los conceptos básicos de la API de Workfront, qué es, cómo funciona y por qué importa, antes de profundizar en sus cinco casos de uso favoritos que muestran su poder en acción.

>[!VIDEO](https://video.tv.adobe.com/v/3492587/?learn=on&enablevpops)

Desde conocimientos básicos hasta aplicaciones reales, esta sesión le proporcionará una ruta clara y accesible al mundo de las API de Workfront, guiada por un experto que trabaja con ellas todos los días. Tanto si es nuevo en las API como si desea ampliar su conjunto de herramientas técnicas, esta sesión le ayudará a conectar los puntos entre concepto y capacidad.
