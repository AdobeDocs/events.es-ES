# Vídeo de información general sobre la versión 2026.01.0 de AEM

Las funciones de esta versión se centran en las innovaciones de Experience Manager Sites, Forms, Assets y Cloud Foundation, e incluyen las siguientes:

* AEM Sites
* Actualizaciones de Edge Delivery

* AEM Assets
* Búsqueda por IA (multilingüe, semántico)

* AEM Forms
* IU asociada de Interactive Communications

* AEM Cloud Foundation
* Servidor MCP
