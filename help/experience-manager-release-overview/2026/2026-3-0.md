# Vídeo de información general sobre la versión 2026.03.0 de AEM

>[!VIDEO](https://video.tv.adobe.com/v/3483060/?learn=on&enablevpops)

Las funciones de esta versión se centran en las innovaciones de Experience Manager Sites, Forms, Assets y Cloud Foundation, e incluyen las siguientes:

* Sitios
   * Asistencia para actualizaciones e inicios del agente de producción de Experience

* Formularios
   * Compatibilidad del flujo de trabajo con el editor IC
   * Compatibilidad del editor de temáticas con componentes principales
   * Compatibilidad con versiones para Forms

* Recursos
   * Asesor de contenido de AEM: GA con aplicaciones de Adobe
   * Conector de Microsoft Office (PowerPoint, Word, Excel)

* Cloud Foundation
   * Agente de administración: administración de marcas
   * Permisos de auditoría en la IU táctil
   * Función de cierre de sesión OIDC
   * Pausar actualizaciones de mantenimiento

* Cloud Manager
   * Servidor MCP de Cloud Manager
