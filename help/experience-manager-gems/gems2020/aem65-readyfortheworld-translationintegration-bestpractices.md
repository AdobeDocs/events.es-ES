# AEM 6.5 listo para el mundo: integración de traducciones y prácticas recomendadas{#aem-ready-for-the-world-translation-integration-best-practices}

Esta sesión presenta las funciones de traducción actuales de AEM y las prácticas recomendadas para administrar sitios web multilingües. Aprenderá a estructurar sitios web globales y a utilizar copias de idioma, Live Copies y MSM. La sesión también explica cómo lograr una traducción de contenido más fácil, rápida y manejable mediante funciones avanzadas.
