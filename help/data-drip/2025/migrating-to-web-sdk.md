# Goteo de datos: ventajas de migrar de un sistema antiguo a Web SDK

Únase a nosotros en este seminario web de goteo de datos, donde el asesor técnico del cliente de Adobe, Arindam Agarwal, nos guiará a través de los beneficios sistémicos del kit de desarrollo de software web de Adobe. Adobe Web SDK es una solución del lado del cliente que aprovecha sus servicios a través de Adobe Experience Platform Edge Network, lo que crea ventajas competitivas sostenidas, reduce los costes de funcionamiento y mejora el rendimiento en comparación con las bibliotecas más antiguas. Arindam proporcionará una descripción detallada de Web SDK, sus características y ventajas como: - Facilidad de implementación - Integración simplificada - Habilitación de seguimiento más inteligente - Reducciones de responsabilidad - Reenvío de eventos - Flujos de tráfico del sitio web

>[!VIDEO](https://video.tv.adobe.com/v/3476271/?learn=on&enablevpops)

## Desbloquear el poder de Adobe Web SDK

Descubra cómo la migración a Adobe Web SDK transforma la recopilación y personalización de datos digitales. Este enfoque moderno optimiza los flujos de trabajo y mejora las pruebas de futuro de su organización.

* **Recopilación de datos unificada** Reemplace varias bibliotecas con una sola SDK ligera, lo que reduce el peso de la página y el mantenimiento.
* **Integración de Edge Network** enrute todos los datos a través de los servidores distribuidos de Adobe para una entrega eficiente a las plataformas de análisis, personalización y de terceros.
* **Rendimiento y cumplimiento** Disfrute de sitios web más rápidos, mejor control y funciones de privacidad integradas, como administración de consentimiento.
* **Implementación simplificada** Migre con una planificación clara, asignación de esquemas y herramientas de depuración sólidas.

El uso de estas funciones permite disponer de datos más limpios, un cumplimiento más estricto y experiencias digitales escalables.

## Recopilación de datos unificada explicada

Adobe Web SDK consolida varias bibliotecas de seguimiento (Analytics, Target y Audience Manager) en un script alloy.js unificado. Esta sola SDK:

* Envía todos los datos en una llamada a la red perimetral de, lo que reduce las solicitudes de red y los tiempos de carga de las páginas.
* Gestiona la recopilación y personalización de datos, lo que elimina la necesidad de solicitudes independientes.
* Se integra con la red perimetral de Adobe, que enruta los datos a soluciones configuradas y destinos de terceros.
* Admite esquemas XDM (Experience Data Model) estandarizados para datos coherentes en todas las plataformas.

Este enfoque simplifica el mantenimiento, mejora el rendimiento y permite una colaboración perfecta entre los equipos.
