# Migración a AEM Cloud Service

En esta sesión, se le presentarán las técnicas y herramientas disponibles para facilitar la migración a AEM Cloud Service. Desde la herramienta de transferencia de contenido hasta la nueva Cloud Acceleration Manager, utilizada junto con el Analizador de prácticas recomendadas para evaluar su entorno, no se pierda esta sesión de habilitación para asegurarse de que está listo para un Cloud Service de go-live on exitoso.
