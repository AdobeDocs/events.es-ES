# Sesión de campeón de AEM de inquilinos múltiples

La sesión se centrará en cómo pueden existir varios inquilinos en la misma instancia de AEM as a Cloud Service. El enfoque está en el uso de Adobe Cloud Manager para implementar varios repositorios de código en la misma instancia de AEM. Esto incluye la configuración de diferentes repositorios de Git, así como el uso de submódulos de Git.
